# delaunay-triang-rs

Delaunay triangulation and Voronoi diagrams in pure Rust.

## Features

- **Bowyer-Watson** — Incremental Delaunay triangulation
- **Voronoi** — Voronoi diagram from Delaunay dual
- **Edge Flip** — Edge flip algorithm for Delaunay property
- **Quad-Edge** — Guibas-Stolfi quad-edge data structure

No external dependencies. Pure `std` Rust.
